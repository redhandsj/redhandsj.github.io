まえがき#

　こんにちわ。今回は、小勢力回路・接地工事・漏電遮断器の施設と省略・住宅の屋内電路の対地電圧制限・臨時配線について見ていきたいと思います。
　では、どうぞ。

小勢力回路#

小勢力回路の施工ポイント#

　家の中には、エアコンのような大きな電気を使う機器もあれば、玄関のベルやチャイムのような小さな電気を使う機器もあります。そんな小さな電気を使う機器に接続する為、『絶縁変圧器を用いて60[V]以下に電圧を下げた回路』を『小勢力回路』といいます。
　『絶縁変圧器』とは、一次側巻線と二次側巻線が電気的に絶縁された変圧器です。二次側電路は大地から絶縁されるので、比較的安全な回路です。ＪＩＳ記号の総称は『小形変圧器』です。

接地工事#

接地工事の重要ポイント#

　これまでに書いてきた様々な工事に必ず『接地』という言葉が出てきたはずです。作業的に言えば、『電気設備と大地を電気的に接続する』ことです。何故やるのかというと、異常電圧抑制・漏電による感電防止・漏電遮断器の確実動作などを目的としています。
　ひとことで『接地』といってもＡ種～Ｄ種まで４種類の接地工事があります。詳しくはおいおいやっていきますが、低圧機器の場合は、Ｃ種とＤ種になります。特に使用電圧が300[V]以下の場合は、Ｄ種になります。

接地工事が省略できる場合#

　基本的には安全の為、全ての場合に接地は行った方がよいのですが省略可能な場合があるので描きます。

  • 対地電圧150[V]以下の機器を、乾燥した場所に施設したとき
  • 低圧用機器を、乾燥した木製の床など、絶縁性のものの上で取り扱うよう施設した時
  • 水気のない場所に施設した機器に電源を供給する電路に漏電遮断器（感度電流15[mA]以下、動作時間0.1[s]以内）を施設した時
  • 電気用品安全法の適用を受ける二重絶縁構造の機器を施設する時

地絡と短絡#

　『地絡』は、電気機器や配線から漏れた電気が何かの伝導体を伝って大地に電気が流れる事。
　『短絡（ショート）』は、電路間の絶縁の効果が下がって電流が漏れ流れ出る事。

漏電遮断器の施設と省略#

漏電遮断器が省略できる場合#

　『漏電』とは読んで字のごとく電気が漏れる事です。通常絶縁に問題なければ、普通に電気機器が使用できますが、絶縁が劣化した場合など、不意に電路から漏れ出した電気で感電や火災の事故が起こる可能性があります。そこで、漏電した場合に自動的に電気を遮断しないといけません。それが『漏電遮断器』の役割です。
　次の条件が揃った場合ではその電路に必ず漏電遮断器を設置しなければなりません。

  • 金属製の外箱がある
  • 使用電圧が60[V]を超える低電圧機器を使用
  • 簡易接触防護措置を施さない
    　また、逆に省略できる場合などを詳しく以下に描いておきます。

住宅の屋内電路の対地電圧制限#

屋内電路の対地電圧を300V以下にできる場合#

　一般の住宅の『対地電圧は、150[V]以下』と規定されています。が、大きな電力を使用する機器を使用したく（定格消費電力が２[kW]以上の電気機器など）電力が足りない場合は、以下の４つの規定に準じれば、『対地電圧は、300[V]以下』で設置する事ができます。

  • 簡易接触防護措置を施す
  • 電気機器は屋内配線と直接接続する（コンセントは使えない）
  • 専用の開閉器及び買い電流遮断器を施設する（専用回路にする）
  • 漏電遮断器を施設する

臨時配線#

臨時配線の使用は原則工事完了後４か月以内#

　祭りの提灯や工事中の照明など、一時的に接地する配線に関しては、施工規定を緩和して施設する事ができます。
　まず『使用電圧が300[V]以下』であれば、屋内のがいし引き工事、または、絶縁電線による配線を臨時配線として４か月以内に限り使用できます。また、『使用電圧が 150[V]以下』であれば、屋外で絶縁電線による配線を臨時配線として４か月以内使用可能です。
　そして、コンクリートにケーブルを直接埋没する場合は、低圧分岐回路で、漏洩遮断器と専用開閉器、過電集遮断器を施設すれば、１年以内に限り使用可能です。

接触防護措置#

　技術基準などで時々登場する『接触防護措置』とは何でしょう？
　簡単にいうと、『人が触れない様に接地する』という意味です。類似語でいうと『簡易接触防護措置』は『容易に人が触れない様に接地する』になります。まぁ、『簡易』の場合は、がんばったら触れるんだけど、簡単には触れないように設置してくださいという事ですね
　ちなみに、金属製の場合、防護したものが、電気的に接続の可能性があれば、それは『簡易接触防護措置』を施したとはいいません。

施設場所の定義#

　これまで、施設場所の用語として『水気のある場所』や『乾燥した場所』など曖昧な表現が使われています。が、これらももう少し詳しく以下に描いておきます。

  • 乾燥した場所 … 湿気の多い場所及び水気のある場所以外
  • 湿気の多い場所 … 水蒸気が充満する場所、または、湿度が著しく高い場所
  • 水気のある場所 … 水を扱う場所、もしくは雨線外、その他の水滴が飛散する場所、または、常時水が漏出し、もしくは、結露する場所。
  • 高温場所 … 周囲温度が通常の使用状態において、30°を超える場所
  • 露出場所（展開した場所） … 点検できない隠ぺい場所及び点検できる隠ぺい場所以外の場所
  • 点検できる隠ぺい場所 … 点検口がある天井裏、二重構造の床下、戸棚または押入れなど、容易に電気設備に接近し、または点検できる場所
  • 点検できない隠ぺい場所 … 点検口がない天井ふところ、壁内またはコンクリート床内など、工作物を破壊しなければ電気設備に接近、または接近できない場所
  • 雨線内 … 屋外及び屋側（建造物の屋外側面）において、のき、ひさしなどの先端から鉛直線に対し、建造物の方向に４５度の角度で下方に引いた線より内側の部分で、通常の降雨状態において雨がかからない場所
  • 雨線外 … 屋外および屋側において、雨線内以外の場所（雨のかかる場所）
