まえがき#

　こんにちわ。今回は、電験の４科目の１つ理論の基礎知識を扱います。様々な項目の共通した知識を洗い出したいと思います。では、どうぞ。

理論の全体図#

　理論科目の全体図です。

直流回路#

抵抗の合成#

　まず、電池との接続には『直流』と『交流』があります。これらの２つのつなぎ方によって、抵抗値の合計の計算が異なります。

直流回路の法則#

キルヒホッフの第一法則#

　ある点に流れ込む電流の和と、そこから流れ出る電流の和は等しい。

キルヒホッフの第二法則#

　回路を１周してたどった時、その通り道の起電力の総和は電圧降下の総和となる

重ね合わせの理#

　複数の電源がある時に、１つずつの回路に分解してから計算しても変わらない

テブナンの定理#

　求めたい電流がある時に、それ以外の複雑な回路をまとめて計算する定理

ブリッジ回路#

　直並列回路の中間点を橋渡ししている回路。斜めに向かい合った抵抗をかけた値が等しい時、橋の部分には電流が流れない（ブリッジの平衡条件）

静電気#

静電気とは#

　『静電気』は物質に帯びた（帯電）ままとどまっている電気。『帯電』とは、正電荷か負電荷のどちらかの電荷が多い状態であること。導体に電荷を近づけると電荷と近い側に、反対符号を持つ電荷が移動されます（静電誘導）。

クーロンの法則#

　２つの電荷の間に働く静電力を求める法則

電界と電束#

　『電界』とは、静電力が働く空間のことです。『電界の強さ』は電界中の正電荷を置いたとき、これに働く静電力の大きさと向きです。これらの電界の様子を視覚的に見る為に、『電気力線』で描きます。

　『電束』とは、電界の状況を示す電気の束を言います。

コンデンサ#

　電源につなげて電気を蓄える事ができる電気回路の部品をコンデンサといいます。基本構造は、絶縁体（誘導体）を導体で挟みます。蓄える電荷の量は、以下の式で求められます。

  • 蓄積電荷Q[C] ＝ 静電容量C[F] × 電圧V[V]
    　ここで『誘電率』は、誘電体の電化蓄積のしやすさを示す定数です。
    　あと、『比誘電率』は、誘電率が真空の誘電率の何倍を示す指標です。

電磁力#

磁界とは#

　磁石の特性である磁力の影響が及ぶ空間が磁界です。磁石の性質を『磁性』といい、磁石の両端が最も磁性が強く、この部分の事を『磁極』といいます。磁気を帯びた物質に磁石と反対の磁極が現れる現象を『磁気誘導』といいます。

　磁界を視覚化するのには『磁力線』を描きます。

　この磁力線の束が『磁束』です。『磁束』と垂直な１㎡の面を通る磁束の量を『磁束密度』といいます。

電磁力とは#

　電流が流れる導体の周りには次回が発生します。電流の向きにより磁界の向きも定まります（右ねじの法則）。

　『電磁力』とは、電流が磁界から受ける力です。

　電磁力の方向は磁界と電流の方向で定まります（フレミング左手の法則）。

　磁界密度と導体の長さ、電流が分かると電磁力の大きさを求める事ができます。

トルク#

　『トルク』とは、回転軸を中心に働く回転力のことです。

電磁誘導#

　コイルに磁石を近づけたり遠ざけたりすると電流が流れます。しかし、コイルと磁石を固定すると電流は流れません。要するに、磁石とコイルの距離が変化している間だけ、電流が流れるという事になります。言いまえると、コイルと貫く磁束が時間的に変化すると、コイルに起電力が発生する現象が『電磁誘導』といいます。この時、磁石の動きが速いほど、コイルに生じる誘導起電力が大きくなります。

  • ファラデーの法則 … 誘導起電力の大きさはコイル内部を貫く磁束の変化に比例する
  • レンツの法則 … 誘導起電力の向きは、コイル内の磁束変化を妨げる向きに生じる
  • フレミング右手の法則 … 誘導起電力の発生による、誘導電流が流れる向き

　『誘導起電力』の大きさは、磁束密度と導体の長さ、導体の移動速度から導くことができます。

  • 自己誘導 … コイルに流れる電流が変化すると、磁束がコイルに起電力を発生させる
  • 自己誘導起電力 … 自己誘導で発生した起電力
  • 自己インダクタンス … コイルの自己誘導しやすさを示す定数

交流回路#

正弦波交流#

　交流は三角関数のsinを使って表せるので、『正弦波交流』といいます。

サイクル#

　『サイクル』は、ある状態から一定の変化を経て、再び元の状態に戻ってくる過程をいいます。

  • 周期 … １サイクルにかかる時間
  • 周波数 … １秒間に繰り返すサイクル回数
    　交流は周波数が時系列に変化している為、直流のように〇〇ボルトという言い方ができません。その為、最大値（波形の最大）、平均値（交流の半周についての平均値）、実効値（交流と同等の直流の値）などで表します。

位相#

　同じ周期であっても、横（時系列）にずれている場合、このずれを『位相』という。

その他単語#

  • インピーダンス … 電流の流れを妨げるもの
  • 皮相電力 … 電力 × 電圧
  • 有効電力 … 電流と電圧の位相差を考慮した有効な電力
  • 力率 … 有効電力と皮相電力の割合（有効電力／皮相電力）
  • 三相交流回路 … 交流回路の位相を120度ずつずらして３つ組み合わせた回路

あとがき#

　様々な状態の電気の考え方が理論の基礎となります。今回の内容は、今後の土台となるので、問題を解いたりして何度もみて覚えましょう。
　わーわーゆうております、お時間です、また次回。
