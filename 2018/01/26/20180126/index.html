電気関係の法規#

そもそも法律とは#

　一言『法』と一言で言っても、その中で法の構成が決められています。『部品としては『憲法』『法律』『命令』『条例』があります。法律といっても国会で決められるものばかりでなく、国会以外の行政機関が決めている『省令』というものもあります。

　法律は、第１条、第２条という『条文』から構成されています。その中に第１項、第２項などの『項』があり、さらにその中で細かく列挙する為に、『号』を使用する場合があります。
　法律用語には注意する単語があります。電験では数字の境界を示す以下の言葉に注意しましょう。

  • 「以上」 … 基準を含んでよれより多い
  • 「超える」 … 基準を含まずそれより多い
  • 「以下」 … 基準を含んでそれより少ない。
  • 「未満」 … 基準を含まずそれより少ない。

　２つの事象を並列した場合、「OR」と「AND」で以下のような表現になります。

電気関係の法律#

　主なものを列挙します。いずれも試験範囲になります。

  • 電気事業法（最重要）
  • 電気用品安全法
  • 電気工事士法
  • 電気工事業法

　『電気事業法』（1964年施行）は、電気事業についての基本的な法律で、電気工作物の保守・運用について定められています。
　『電位用品安全法』は、電気用品の障害防止の為、製造・販売の規制を定めています。「電気用品」とは、電気工作物のまわりで使用される機会・器具・材料のことです。危険度が高いものは、「特定電気用品」といいます。

　『電気工事士法』は、電気工事に携わる人の必要な資格や義務などを定めています。

電気事業法#

　第１条「電気事業の健全な発展を図るとともに、電気工作物の工事、維持及び運用を規制する事によって、公共の安全を確保し、環境の保全を図ること」を目的とします。
　ここでいう「電気工作物」とは、発電・変電・送電・配電などで使用する機械を指します。分類すると以下のようになります。

  • 事業用電気工作物
    • 電気事業用電気工作物
    • 自家用電気工作物
  • 一般用電気工作物
    　『電気事業者』の義務として、以下の義務があります
  • 電圧および周波数の維持
  • 事故の報告
    　電圧の維持に関しては、維持すべき電圧は以下になります。
標準電圧 維持すべき値
100[V] 101±6[V]
200[V] 202±20[V]

　さらに、維持すべき周波数は、「標準周波数」（50[Hz]または60[Hz]）です。
　事業用工作物を設置する者は、以下のような義務があります。

  • 電気工作物を技術基準に適合するように維持
  • 電気主任技術者を選任
  • 保安規定を定め、届け出する
    • 「保安規定」とは、職務・組織・保安・教育・巡視・点検・検査・運転・操作・記録などを指す。

　また、選任する電気主任技術者の職務と範囲は以下。

  • 第１種 … 全ての事業用電気工作物
  • 第２種 … 電圧が17万V未満の事業用電気工作物
  • 第３種 … 電圧が5万V未満の事業用電気工作物（出力5000kW以上の発電所を除く）

電気設備技術基準#

技術基準とは？#

　電気事業法に定めてある工事や保守についての基準です。
　『電気設備技術基準』（電技）とは、電気設備の工事や保守についての技術標準を定めた省令です。
　『電気設備技術基準の解釈』（解釈）とは、「電技」の具体的な内容を定めたものです。
　以下に主な用語の定義を書いておきます。

  • 電路 … 通常の使用状態で電気が通じているところ
  • 電気機械器具 … 電路を構成する機械器具
  • 発電所 … 電気を発生させるところ
  • 変電所 … 構外から伝送される電気を変成する所であって、構外に伝送するもの
  • 電線 … 強電流電気の伝送に使用する電気導体全般
  • 配線 … 電気使用場所において施設する電線

絶縁抵抗と絶縁耐力試験#

　『絶縁』とは、ある２か所の間で電気抵抗が大きく、電圧をかけても電流が流れない状態です。『電技』で定められている電圧の区分を以下に規制される。

  • 低圧 … 直流は、750[V]以下、交流は、600[V]以下
  • 高圧 … 7000[V]以下
  • 特別高圧 … 7000[v]超
    　屋内配線の『低圧電路』の絶縁抵抗は測定が義務付けられています。また、高圧・特別高圧の電路の絶縁性能は絶縁耐力試験を用います。これは、一定の電圧を加えて、絶縁破壊が起きないかを調べます。

接地工事#

　『接地』とは、電気機器を導線で大地につなげる事です。落雷・故障で電気が漏れた場合でも接地をしていれば感電や火災を防止できます。

　接地工事の種類を以下に書きます。

種類 設置する場所
A種接地工事 高圧・特別高圧機器の鉄台、外箱
B種接地工事 高圧・特別高圧を低圧に変圧する変圧器の低圧側
C種接地工事 300Vを超える低圧機器の鉄台、外箱
D種接地工事 300V以下の低圧機器の鉄台、外箱

電気施設管理#

　電気設備を運転する際に、その設備の機能を十分に発揮できるように関する事です。
　『需要率』とは、負荷設備の容量の合計に占める需要電力の割合です。これは、電気設備のすべてをいっぺんに使用する事はあまりなく、必要な電力の最大値（最大需要電力）が設備の合計容量よりは小さくなるのでその割合を求めるものです。

　『負荷率』とは、時間帯・季節によって変化する電力の消費の割合を表したものです。

　『不等率』は、１台の変圧器で複数の需要家に電気を供給する場合、需要電力を合成したものと各需要家ごとの合成需要電力の割合の値です。

　『力率』とは、有効電力と皮相電力の比率です。

　『力率の改善』は、角度を少なくしていけば、ロス（無効電力）がなくなります。
