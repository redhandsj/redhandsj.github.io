シンプルテンプレートの特徴

SEO,見た目,編集のしやすさを追及したテンプレートになっています。 シンプルな構造はSEOに理解されやすく、ブラウザの仕様変更にも対応しやすくなります。 シンプルすぎる場合はJQueryとCSS3を使用したバージョンも用意していますので、そちらもご覧ください。 またカスタマズも用意しているので必要なものだけ取り込む事も可能です。MovableType,WordPress版も有償ですが用意しています。

テンプレートの仕様

全体幅は960pxでメイン:630px,サイド:290px。両サイド10pxとメインとサイドに20pxの余白です。本文は20pxの余白を設けて590pxです。 幅はCSSで1箇所しか指定していないので容易に変更できます。 H1はヘッダー右上端。H2はメインタイトルタグです。

h3タグは文章内の小タイトルに

storongタグは強調表示で本文に抑揚をつけましょう。
文章の引用はblockquoteで囲んで表示
画像左寄せ 画像右寄せ よってもメリハリをつけることができます。

使用しているhtml5追加タグ

!DOCTYPE html / html / header / nav / article / aside / footerの構造タグのみを使用しています。 IE6-9,FireFox ,opearaで表示確認をしています。IE6-9は下記タグをヘッダーに入れる事でレイアウトを維持しています。
<script src="http://html5shiv.googlecode.com/svn/trunk/html5.js" type="text/javascript"></script>

カラー展開

色はcssで10箇所指定しています。 容易に自分好みの色に変える事ができます。
