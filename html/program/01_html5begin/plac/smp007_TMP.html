デラックステンプレートの特徴

基本構造はシンプルテンプレートと同様ですが、JavaScriptとCSS3を使いプルダウン、開閉スクリプト、タブスイッチ、グラデーションと角丸を追加しています。 続きを読む
シンプルな構造はSEOに理解されやすく、ブラウザの仕様変更にも対応しやすくなります。 IEも極力スクリプトを使って表示できるようにしていますがIE6だとが表示できず、ブラウザのアップグレードを促すメッセージを表示するようにしています。（削除可） カスタマイズをおこなう際に難しいと感じる場合、シンプル版から取り入れたいものをカスタマイズ版から MovableType,WordPress版も有償ですが用意しています。

テンプレートの仕様

全体幅960px,メイン630px,サイド290px。幅はCSSで1箇所しか指定していないので容易に変更できます。 続きを読む
両サイド10pxとメインとサイドに20pxの余白です。本文は20pxの余白を設けて590pxです。 H1はヘッダー右上端。H2はメインタイトルタグです。

h3タグは文章内の小タイトルに

storongタグは強調表示で本文に抑揚をつけましょう。
文章の引用はblockquoteで囲んで表示
画像左寄せ 画像右寄せ よってもメリハリをつけることができます。

使用しているhtml5追加タグ

!DOCTYPE html / html / header / nav / article / aside / footerの構造タグのみを使用。
IE6-9,FireFox ,opearaで表示確認をしています。 続きを読む
IE6-9は下記タグをヘッダーに入れる事でレイアウトを維持しています。
<script src="http://html5shiv.googlecode.com/svn/trunk/html5.js" type="text/javascript"></script>

カラー展開

画像は一点も使わずCSSのみで装飾しています。CSSでも3色10箇所しか指定していないので、容易に自分好みの色に変える事ができます。 続きを読む
