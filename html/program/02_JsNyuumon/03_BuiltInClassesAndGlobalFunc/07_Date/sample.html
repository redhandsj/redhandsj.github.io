Dateクラス

コンストラクタ関数

toString / toDateString / toTimeStringメソッド

toLocaleString / toLocaleDateString / toLocaleTimeStringメソッド

toUTCString / toGMTStringメソッド

  • 対象のDateクラスのオブジェクトを文字列に変換した結果を返します。この時、タイムゾーンとして現地時間ではなくUTCを使います。

    valueOfメソッド

  • 対象のDateクラスのオブジェクトをミリ秒形式に変換した値を返します。

    UTCメソッド

  • UTCタイムゾーンで指定した日付/時刻の値をミリ秒形式の値に変換した値を返します。

    parseメソッド

  • 文字列として与えられた値から日付/時刻の値を解析し、取得した日付/時刻の値をミリ秒形式の値に変換した値を返します。

    getFullYear / getMonth / getDate / getDayメソッド

  • Dateクラスのオブジェクトから日付に関する年/月/日/曜日の値を取得するメソッドがそれぞれ用意されています。

    getUTCFullYear / getUTCMonth / getUTCDate / getUTCDayメソッド

  • オブジェクトから日付に関する年/月/日/曜日の値を取得するメソッドがそれぞれ用意されています。このページで解説しているメソッドはタイムゾーンがUTCでの値を取得します。

    getHours / getMinutes / getSeconds / getMillisecondsメソッド

  • オブジェクトから時刻に関する時/分/秒/ミリ秒の値を取得するメソッドがそれぞれ用意されています。

    getUTCHours / getUTCMinutes / getUTCSeconds / getUTCMillisecondsメソッド

  • オブジェクトから時刻に関する時/分/秒/ミリ秒の値を取得するメソッドがそれぞれ用意されています。このページで解説しているメソッドはタイムゾーンがUTCでの値を取得します。

    getTimezoneOffsetメソッド

  • UTC又はGMTと現地時間との時間差を返します。

    getTimeメソッド

  • 対象のDateクラスのオブジェクトをミリ秒形式に変換した値を返します。

    setFullYear / setMonth / setDateメソッド

  • Dateクラスのオブジェクトに対して日付に関する年/月/日の値を設定するメソッドがそれぞれ用意されています。

    setUTCFullYear / setUTCMonth / setUTCDateメソッド

  • Dateクラスのオブジェクトに対して日付に関する年/月/日の値を設定するメソッドがそれぞれ用意されています。

    setHours / setMinutes / setSeconds / setMillisecondsメソッド

  • Dateクラスのオブジェクトに対して日付に関する年/月/日の値を設定するメソッドがそれぞれ用意されています。

    setUTCHours / setUTCMinutes / setUTCSeconds / setUTCMillisecondsメソッド

  • Dateクラスのオブジェクトに対して時刻に関する時/分/秒/ミリ秒の値を設定するメソッドがそれぞれ用意されています。

    setTimeメソッド

  • 対象のDateクラスのオブジェクトをミリ秒形式の値で設定します。
