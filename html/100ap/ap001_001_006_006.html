6.6　CPUの高速化技術

概要

■パイプライン処理
命令内での各ステージを並行処理する事で処理時間を短縮
■スーパーパイプライン
パイプラインで並行していたステージを更に細かく分けて並行処理する
■スーパースカラ
パイプラインを複数並行して実施する
■VLIW(Very Long Instruction Word)
複数動作を纏めて１命令で処理する
■CISC(Complex Instruction Set Computer)
CPUに複雑で高速な命令を持たせる
■RISC(Reduced Instruction Set Computer)
CPU内の単純な命令の組合せで処理を行う
